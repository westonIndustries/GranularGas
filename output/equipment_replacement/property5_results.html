Interpretation

  • Monotonicity: The survival function S(t) represents the probability that equipment survives to age t. It must be monotonically decreasing (or flat) because equipment cannot "un-fail".
  • S(0) = 1.0: New equipment (age 0) always survives, so S(0) must equal 1.0.
  • Replacement Probability Bounds: The replacement probability P(t) represents the conditional probability that equipment fails in the next year. It must be in [0, 1] to be a valid probability.
  • Shape Parameter β: Higher β values (e.g., 3.0 for HVAC) indicate concentrated replacement windows (most equipment fails around the median life). Lower β values (e.g., 2.0 for fireplaces) indicate more gradual failure.
